# Task 1: Basic Data Retrieval & Filtering (PostgreSQL)

## Description
This repository contains SQL scripts for Task 1 of the SQL Internship.
The task focuses on basic data retrieval, filtering, sorting, pagination,
and validation using PostgreSQL.

## Database Used
- PostgreSQL
- Table: employees

## Topics Covered
- SELECT specific columns
- WHERE clause (filters, BETWEEN, LIKE, NULL handling)
- ORDER BY (single & multiple columns)
- LIMIT and OFFSET
- Data validation using COUNT, MIN, MAX

## How to Run
1. Create the employees table using schema/employees_table.sql
2. Insert sample data
3. Execute queries in the queries/ folder in order
